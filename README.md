# Incorta Schema Explorer

A modern web tool for exploring, comparing, and validating Incorta schema exports. All processing is 100% local—no data leaves your machine.

## Features
- **Explore**: Load a single Incorta schema export (ZIP or XML) to browse tables and columns interactively.
- **Compare**: Load two schema files to view differences between versions or tenants.
- **Filter**: Powerful sidebar filters for table type, data source, column type, and more.
- **Diff Report**: Visualize changes in tables, columns, and configurations.
- **Privacy**: All processing is done in your browser; no uploads or external servers.

## Usage
1. **Open the app** (index.html) in your browser.
2. **Load schema files**:
	- File A: Reference schema (ZIP or XML)
	- File B: (Optional) Second schema for comparison
3. **Browse and filter** tables, columns, and differences.

## File Types Supported
- Incorta schema ZIP exports
- Incorta schema XML files

## How to Run Locally
- Just open `index.html` in any modern web browser (Chrome, Edge, Firefox, etc.).
- No installation or server required.
